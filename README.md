# react-native-glow-tour

Beautiful spotlight tour for React Native — Reanimated v4, Fabric ready, zero data coupling, works with Expo

## Installation


```sh
npm install react-native-glow-tour
```


## Usage


```js
import { multiply } from 'react-native-glow-tour';

// ...

const result = multiply(3, 7);
```
